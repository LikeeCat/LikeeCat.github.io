Xcode快捷键

2019-05-30
tip

编辑的快捷键

快捷键名称 作用
command + ↑ 到达文件头部
command + ↓ 到达文件尾部
command + [ 向左移动代码
command + ] 向右移动代码
option + command + [ 向上移动代码
option + ↑ 向上移动一行代码
option + ↓ 向下移动一行代码
option + command + ] 向下移动代码
command + / 快速注释或者取消注释
Ctrl + A 移动光标到本行行首
Ctrl + E 移动光标到本行行尾
option + → 移动光标到下一个单词
option + ← 移动光标到上一个单词
