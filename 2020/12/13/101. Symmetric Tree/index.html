101. Symmetric Tree

2020-12-13
leetcode

题目

一些思考

  1. 从给出的内容来看,看这个二叉树是不是一个镜像的二叉树,即它的左子树与右子树之间是对称的
  2. 那么他的左子树和右子树是镜像的,对于一个三层的二叉树来说就是,左子树的左侧节点和右子树的右侧节点是一致的,左子树的右侧节点和右子树的左侧节点是一致的

思路

代码

/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     public var val: Int
 *     public var left: TreeNode?
 *     public var right: TreeNode?
 *     public init() { self.val = 0; self.left = nil; self.right = nil; }
 *     public init(_ val: Int) { self.val = val; self.left = nil; self.right = nil; }
 *     public init(_ val: Int, _ left: TreeNode?, _ right: TreeNode?) {
 *         self.val = val
 *         self.left = left
 *         self.right = right
 *     }
 * }
 */


 class Solution {

  func isMirror(_ left:TreeNode?,_ right:TreeNode?) -> Bool {
    if left == nil && right == nil {
        return  true
    }
    if left == nil || right == nil {
        return false
    }
    return (left?.val == right?.val) && isMirror(left?.right, right?.left) && isMirror(left?.left, right?.right)

    }

    func isSymmetric(_ root: TreeNode?) -> Bool {
            return isMirror(root,root)
    }
}

注

这道题的重点还是要理解两个子树之间的关系,即怎样的两个子树即可互为镜像

1. Their two roots have the same value
2. The right subtree of each tree is a mirror reflection of the other tree
